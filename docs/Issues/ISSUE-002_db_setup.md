# ISSUE-002 DB接続・マイグレーション設定

## 親 Issue
ISSUE-001

## 概要
PostgreSQLへの接続とGORMマイグレーションを設定する。アプリ起動時にDBに接続し、全モデルのテーブルを自動生成する。

## 変更ファイル一覧

- `app/src/repository/db.go`（編集）
    - **何を**: GORM PostgreSQL接続の初期化関数とグローバルDatabase変数の定義。全モデルを対象としたAutoMigrateの実行。
    - **なぜ**: DB接続を一元管理し、全レイヤーから参照できるグローバル変数として公開する必要があるため
- `app/src/main.go`（編集）
    - **何を**: アプリ起動時にrepository.InitDB()を呼び出す処理の追加。
    - **なぜ**: サーバー起動前にDBマイグレーションを完了させる必要があるため

## テスト確認項目

- [ ] アプリ起動時にDBに接続できること
- [ ] 全モデルのテーブルが自動生成されること
- [ ] 接続エラー時にアプリが起動しないこと
